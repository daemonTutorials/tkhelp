tkhelp Webinterface

Das tkhelp Webinterface ist eine Python-Anwendung, eher gesagt ein CGIHTTPServer in Python geschrieben.

Einerseits besteht sie aus Python-Skripten(Stichwort WSGI) und JavaScript. Da wir Ladezeiten verkleinern wollen, nutzen wir viel AJAX, darum ist JavaScript Pflicht!
